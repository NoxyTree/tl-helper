## The project in one paragraph

TL-Helper turns installed Throne and Liberty game archives (Steam build
`24118850`, game `1.431.22.7761`) into decoded, normalized, verified data and
web tools. Code lives in `D:\TL_Helper`; bulk data in `D:\TL_Data`
(`$env:TL_DATA_ROOT`). The pipeline — collector → decoder → SQLite warehouse
(85,099 records, 38 tables) → browser projections — is orchestrated by
`node scripts\update-tl-helper.mjs` and fully verified. On top sit the Armory,
tracker, achievements, static build calculator (BuildSnapshot v1 boundary),
a 293k-row stat-source index, and the in-progress Combat Simulator.

## Today's two evidence gains

1. **Questlog contest curves captured** (`2332b5b`,
   `questlog-contest-curve-evidence.md`). Their public simulator models every
   stat contest (hit/evasion, crit/endurance, heavy chance/heavy evasion,
   skill damage boost/resistance) as
   `chance% = max(0, A−B)/((A−B)+1000) × 100`, signed-mirrored when the
   attacker leads, per damage type, no base miss. Confirmed by behavioral
   probing and by the formula module in their JS bundle. Precision:
   `derived_community_reference` — usable in the engine ONLY as `modeled`
   with provenance; register rows 3–5 stay calibration-required.

2. **First live calibration session** (`3f5727c`,
   `calibration-findings-2026-07-10.md`; 21 observations + 27 screenshots in
   the store). Results:
   - Tooltip coefficient basis **verified exactly** for all three abilities
     (protocol groups 1–3 complete): `tooltip1 = mul/100`, `tooltip2 = add`.
   - **Rarity tiers are 5-level windows** into the global 21-level table:
     Epic Lv.N = global N+10, Heroic Lv.N = global N+15.
   - **Per-cast Base Damage roll strongly supported** (user hypothesis;
     heals 2,714–3,114 normal, shields 5,003–7,932). Multiplier
     (Healing +6%? ×1.042 received?) and rounding NOT yet discriminated —
     needs protocol groups 6–7 (single-item gear swaps, ≥20 casts, HP
     before/after; user will screen-record next time).
   - **Heavy Attack procs on heals** (player-confirmed, ×2 floating text);
     healing needs a heavy outcome branch in the engine.
   - **Distortion Veil magnitude anomaly:** observations fit neither the
     full +71.47% Shield Health bonus nor no bonus; effective multiplier
     ~+26–49%. Unresolved — do not model DV shield magnitude.

## Rules that must not be relaxed

- Nothing from `unknown-formulas.md` may be silently invented. Every engine
  formula ships as `extracted`/`derived`/`modeled`/`calibrated` with
  provenance, or stays `unsupported` and non-executable.
- Community references (Questlog) never upgrade past `modeled`.
- Calibration observations are immutable; reviewers add conclusions, never
  rewrite raw data.
- Only visible game info, manual input, screenshots/recordings — no memory
  reading, packet capture, or gameplay automation.
- No cloud upload of game data; no off-machine backup without explicit user
  authorization.
